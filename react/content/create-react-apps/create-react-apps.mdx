# Creating a React application

A React app is a React Component that is deployable, and therefore consumable in runtime. That includes full applications as well as micro-frontends.

A Component is made deployable by adding to it a file with a specific app type configuration. An app type determines how that Component (the app) is built and deployed.

### Create a React app component

Run the following, to get a list of all available React Component templates:

```
bit templates teambit.react/react
```

The output list should contain app and non-app React Components templates:

```
..
```

Run the following to create a React app that uses the react-app app type:

```
bit create react-app apps/my-react-app
```
